# Sistema de Gestión de Invernadero 🌱

Este es un proyecto en **Java** que implementa un sistema para administrar y monitorear variables clave en un invernadero, utilizando **PostgreSQL** como base de datos y librerías de conexión **COM** para la interacción con la base de datos.

---

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Java
- **Base de Datos**: PostgreSQL
- **Librerías**:
  - **JDBC**: Para la conexión con la base de datos.
  - **COM**: Librería para gestión de conexión.

---

## 🎯 Funcionalidades Principales

1. **Gestión de Datos del Invernadero**:
   - Registrar, consultar, actualizar y eliminar datos relacionados con cultivos.
2. **Interacción con la Base de Datos**:
   - Implementación de consultas dinámicas para gestionar variables ambientales como temperatura, humedad y luz.
3. **Código Modular**:
   - Separación de responsabilidades en capas para facilitar el mantenimiento.
